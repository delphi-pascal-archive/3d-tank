Файл ReadMe к движку by BoogeMan

Хоть обычно такие файлы никто не читает, я все равно решил его написать...

Заbindиные клавиши:
[W,S] - вперед, назад
[A,D] - влево, вправо
[1,3] - поднять, опустить (пушку)
[f,g] - сделать холм, яму
[y,u] - вращать камеру
[k,l] - поднять, опустить (камеру)
[7,9] - приблизить отдалить (камеру)
[0] - (ноль а не o) привязать камеру к танку, отвязать
